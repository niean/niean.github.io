# 产品需求 - 稳定固化(不频繁变更)

## 1. 极简摘要

niean的个人技术博客，基于 Jekyll + GitHub Pages 构建，记录运维、SRE、架构等领域的实践与思考。

产品定位、体验原则与判断准则见 .harness/prd/01-prd-sense.md。

---

## 2. 页面结构

- 首页：博客文章列表（按时间倒序）
- 文章页：单篇博客文章展示
- 归档页：按时间归档的文章列表
- 分类页：按分类组织的文章列表
- 标签页：按标签组织的文章列表
- 草稿页：未发布的草稿文章列表
- 关于页：作者介绍

---

## 3. 功能需求

### 已实现功能

- 文章发布：Markdown 格式，自动构建部署
- 文章分类：支持多分类
- 文章标签：支持多标签
- 文章归档：按时间归档
- 草稿管理：草稿预览功能
- 分页：文章列表分页（每页16篇）
- RSS订阅：Atom Feed
- 打赏功能：文章底部打赏入口
- 评论功能：文章底部评论区（通过第三方服务）

### 待开发功能

见 .harness/prd/03-prd-specs.md

---

## 4. 非功能与约束

- 部署：GitHub Pages，自动部署
- 域名：blog.niean.name
- HTTPS：GitHub Pages 自动启用
- 响应式：支持移动端访问
- 加载速度：静态页面，无后端依赖
